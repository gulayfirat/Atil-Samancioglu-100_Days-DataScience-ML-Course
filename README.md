## üöÄ Project Goal and Scope

This repository documents a learning journey aimed at acquiring the fundamental knowledge and practical skills necessary in the fields of Data Science and Machine Learning.

* **Fundamentals:** Python programming language and essential data manipulation libraries.
* **Data Analysis & Visualization:** In-depth Exploratory Data Analysis (EDA) using NumPy, Pandas, Matplotlib, and Seaborn.
* **Machine Learning Algorithms:** Practical applications of Supervised (Regression and Classification) and Unsupervised Learning (Clustering) algorithms.
* **Data Preprocessing (Preprocessing):** Techniques for Feature Engineering, Handling Missing Data, Scaling, and Encoding.
* **End-to-End Project:** Reinforcement of learned knowledge through a real-world machine learning project (Diamond Price Prediction).

## üìÅ Folder Structure and Content

The content is logically organized by topic flow as follows:

| Folder Name | Content Description |
| :--- | :--- |
| `PythonIntro` | Introduction to the Python language, data types, loops, functions, and Object-Oriented Programming (OOP) fundamentals. |

| `DiamondPricePredictMLProject` | End-to-end Diamond Price Prediction Machine Learning Project (Model saving and web application integration with **FastAPI**). |

| `assignments` | Assignments and solutions related to various topics covered throughout the course. |
